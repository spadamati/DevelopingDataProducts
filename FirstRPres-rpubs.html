My First ShinyApp Overview

This ShinyApp allows the user to visualize the iris dataset being clustered(using k-means algorithm) interactively. The user has to hit the submit button after selecting his choices for the X and Y-axis variables using the 'up and down arrows' and the desired number of clusters to be created using the slider.The user can also type any text in the textbox and hit the submit button.There are also tabbed panels for viewing the plot,table,summary and notes.

Sample Code and Plot

library(ggplot2)
data(iris)
iris <- iris
ggplot(iris, aes(x = Sepal.Length, y = Sepal.Width, colour = Species)) + geom_point()
